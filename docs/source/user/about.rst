Radio Butler is a simple tool for enhancing the quality of radio broadcast. It is based in radioDNS standard. Here is a simple guide how to use it.

**Overview**

Radio Butler is a hybrid radio solution for those who want to add more depth in their radio broadcasts. Its main idea is simple: with Radio Butler, you can send visual content to your listeners over IP, using open standard technologies supported by broadcasters and automotive manufacturers globally. The content is sent by Visual Content -function, which uses a RadioDNS -standard. RadioDNS Hybrid RadioÂ® is implemented in Audi, BMW, VW and Porsche cars for example.
